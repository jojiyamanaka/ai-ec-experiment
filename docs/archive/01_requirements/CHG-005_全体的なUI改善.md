# CHG-005: 全体的なUI改善（エディトリアルデザイン化）

## 背景・課題
- 現状のUIが無機質で、ファッションECとしてのブランド感やストーリーが伝わらない
- 情報密度が高く余白が少ないため、商品が「在庫」に見え、作品としての価値が伝わりにくい
- システム的な青色や影の多用により、洗練度や高級感が不足している
- タイポグラフィが単調で、雑誌のような視覚的階層や美しさがない
- ヘッダーが主張しすぎており、商品体験の妨げになっている

## ビジネス要件
- ユーザーが雑誌を読むように、ゆったりと商品を眺められること
- ブランドの世界観とストーリーが、UI全体から一貫して感じられること
- 商品一つ一つが「作品」として際立ち、価値が伝わること
- 見出しと本文で書体の役割が明確に分かれ、読みやすく美しいこと
- 文字間が広く取られ、上品で落ち着いた印象を与えること
- モノトーンとニュアンスカラー（グレー・ベージュ系）で統一され、高級感があること
- ヘッダーが透過で控えめであり、スクロール時も商品体験を邪魔しないこと
- 商品画像が縦長で、ファッションアイテムの全身シルエットが美しく見えること
- ホバー時の動きがゆっくりと滑らかで、丁寧に作られた印象を与えること
- 価格表示が控えめで上品であり、商品の価値を損なわないこと
- 商品カードに装飾的な影がなく、シンプルで洗練されていること
- セクション間の余白が十分に取られ、情報が詰まっていないこと

## 受け入れ条件
- 全画面で背景色がストーン系または白で統一され、青色が廃止されていること
- 見出しにセリフ体が採用され、本文にサンセリフ体が使用されていること
- 全体的に文字間が広く取られ、窮屈な印象がないこと
- ヘッダーが透過デザインで、ロゴが中央配置されていること
- 商品カードの画像が 3:4 の縦長比率になっていること
- 商品カードに影が使用されず、境界線が薄いまたはなしであること
- セクション間の垂直方向の余白が従来の2倍以上確保されていること
- 価格が太字ではなく、小さめの細字または中字で表示されていること
- ホバー時のアニメーションがゆっくり（0.7秒程度）であること
- 見やすさ・ブランド感に関するユーザー満足度が向上すること（定性評価）
