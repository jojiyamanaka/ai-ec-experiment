| Core APIエラー | BFFレスポンス | 挙動 |
|--------------|-------------|------|

| 400 Bad Request | 400 | そのまま伝播 ✅ |
| 500 Internal Error | 500 | そのまま伝播 ✅ |

## 4. 補足

- 500伝播の検証は、`/test` 配下に一時的なエラー発生経路を追加して実施後、同ターン内で削除し再ビルド済み
- 現在の稼働構成では `GET /api/test/error` は `404`（経路なし）で、本体機能への影響なし
